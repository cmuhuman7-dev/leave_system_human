📱 วิธีการตั้งค่าแจ้งเตือนใบลาผ่าน LINE Official Account

ทำตามขั้นตอนต่อไปนี้ทีละขั้น เพื่อเชื่อมต่อระบบบันทึกการลากับ LINE

📌 ขั้นตอนที่ 1: สร้าง LINE Official Account
1️⃣ เข้าไปที่ https://manager.line.biz แล้ว Login ด้วยบัญชี LINE ของ HR
2️⃣ คลิก "สร้างบัญชี" → เลือก "LINE Official Account" → กรอกชื่อบัญชี เช่น "HR Human CMU"
3️⃣ เลือกประเภทบัญชีเป็น "ฟรี" (สามารถอัปเกรดได้ภายหลัง)
4️⃣ เสร็จสิ้น จะได้ LINE Official Account พร้อม QR Code ให้พนักงาน Scan เพิ่มเพื่อน
📌 ขั้นตอนที่ 2: เปิดใช้งาน Messaging API
1️⃣ ใน LINE Official Account Manager → ไปที่ การตั้งค่า → Messaging API
2️⃣ คลิก "เปิดใช้งาน Messaging API" → ยืนยัน
3️⃣ ระบบจะสร้าง Provider อัตโนมัติ หรือเลือก Provider ที่มีอยู่แล้ว
4️⃣ เมื่อเปิดใช้งานแล้ว จะเห็นหน้า LINE Developers Console
📌 ขั้นตอนที่ 3: รับ Channel Access Token
1️⃣ เข้า https://developers.line.biz → เลือก Provider → เลือก Channel ที่สร้างไว้
2️⃣ ไปที่แท็บ "Messaging API"
3️⃣ เลื่อนลงหา "Channel access token" → คลิก "Issue" เพื่อสร้าง Token
4️⃣ Copy Token ที่ได้ (ยาวมาก เริ่มต้นด้วย xxxxxxxxxx...) บันทึกไว้ใช้ในขั้นตอนต่อไป
📌 ขั้นตอนที่ 4: รับ User ID ของผู้รับแจ้งเตือน
1️⃣ ผู้รับแจ้งเตือน (เจ้าหน้าที่ HR) ต้อง Add Official Account เป็นเพื่อน ก่อน
2️⃣ ใน LINE Developers Console → Messaging API → เลื่อนลงหา "Your user ID"
3️⃣ Copy User ID (เริ่มต้นด้วย U...) หรือใช้ Group ID หากต้องการส่งเข้ากลุ่ม LINE
4️⃣ สำหรับ Group ID: เพิ่ม Bot เข้า Group LINE แล้วใช้ Webhook เพื่อดึง Group ID
📌 ขั้นตอนที่ 5: เพิ่มโค้ดส่ง LINE ใน Google Apps Script
1️⃣ เปิด Google Apps Script ของโปรเจกต์นี้
2️⃣ ไปที่ Project Settings → Script Properties
3️⃣ เพิ่ม Property:
LINE_TOKEN = [Channel Access Token ที่ได้จากข้อ 3]
LINE_USER_ID = [User ID หรือ Group ID ที่ได้จากข้อ 4]
4️⃣ โค้ดส่ง LINE จะถูกเรียกอัตโนมัติเมื่อบันทึกการลาในระบบ
💻 ฟังก์ชัน sendLineNotification ที่ต้องเพิ่มใน GAS
function sendLineNotification(message) {
  const token = PropertiesService.getScriptProperties()
                  .getProperty('LINE_TOKEN');
  const userId = PropertiesService.getScriptProperties()
                  .getProperty('LINE_USER_ID');
  if (!token || !userId) return;

  const payload = {
    to: userId,
    messages: [{ type: 'text', text: message }]
  };

  UrlFetchApp.fetch('https://api.line.me/v2/bot/message/push', {
    method: 'POST',
    headers: {
      'Content-Type': 'application/json',
      'Authorization': 'Bearer ' + token
    },
    payload: JSON.stringify(payload),
    muteHttpExceptions: true
  });
}

⚠️ ฟังก์ชัน addLeaveRecord ใน GAS จะเรียก sendLineNotification() โดยอัตโนมัติ
